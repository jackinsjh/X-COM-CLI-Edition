☆엑스컴 CLI 에디션을 플레이 해주셔서 감사합니다!☆
본 게임은 X-COM 이라는 턴제 전략 게임 시리즈에서 
영감을 받아 제작하게 되었습니다.
각 팀의 병사들이 차례대로 행동하며 
상대팀의 HP를 모두 깎아 쓰러트리면 승리하게 됩니다.

본 게임을 플레이 할 시에는, 화면 가로 크기를 고정하고, 
세로를 늘려 플레이하는 것을 권장합니다.

☆병사의 능력치 설명
-최대 HP : 병사의 게임 시작시의 HP를 표시합니다.
-현재 HP : 병사의 현재 HP입니다. 데미지를 받으면 감소하며, 
0이 되면 병사가 쓰러지게 됩니다.
-조준 : 병사가 사격을 명중시킬 기본적인 확률입니다.
-화력 : 병사가 사격을 명중시킬 시 입히는 데미지입니다.
-민첩 : 병사들은 민첩 수치가 높은 순서대로 행동하게 됩니다.
-주파수 : 병사의 고유 주파수입니다. 특수 공격과 관련이 있습니다.
-장탄수 : 병사의 남은 장탄수를 나타냅니다. 
0이 되면 재장전을 해야 다시 사격이 가능합니다.

☆맵 설명
-엄폐물과 상하좌우로 바로 옆에 인접하게 된다면, 
엄폐물 너머 방향의 사격에 맞을 확률이 줄어듭니다.
-■ : 완전 엄폐물 - 엄폐물 너머의 사격에 맞을 확률이 40%만큼 줄어듭니다.
-▦ : 반 엄폐물 - 엄폐물 너머의 사격에 맞을 확률이 20%만큼 줄어듭니다.
-□ : 개활지 - 이곳으로 이동이 가능한 빈 칸입니다.
-또한 적이 자신을 사격하는 경우, 자신이 엄폐하지 않은 상태라면, 
치명타에 맞을 확률이 40% 증가합니다.
-엄폐물과 대각선으로 인접하는 것은 엄폐로 인정되지 않습니다.

☆게임의 진행
-민첩 수치가 높은 순서로 병사들이 행동을 개시하게 됩니다.
-모든 병사들이 한 번씩 행동을 완료하면, 라운드가 다음으로 넘어가며, 
다시 전과 같은 순서로 병사들이 각각 행동을 개시합니다.
-한 턴에 병사는 이동, 사격 또는 특수공격을 할 수 있습니다.
-이동 : 병사를 이동시킵니다. 
한 번에 최대 x축과 y축 각각 5만큼 이동이 가능합니다.
엄폐물이나 다른 병사와 겹치는 곳으로 이동할 수는 없습니다.
-사격 : 대상을 지정하고, 사격을 개시합니다.
적을 명중시킬 확률과 치명타 확률이 표시되게 됩니다.
치명타는 평소의 두 배의 데미지를 줍니다.
사격을 하면 장탄수가 1 감소됩니다.
-특수공격 : 특정 대역의 주파수를 과부하시켜, 
해당 주파수를 가진 모든(적과 아군) 병사들에게 3의 피해를 줍니다.
이 공격은 엄폐에 상관없이 반드시 명중하나, 공격력이 낮습니다.
-병사 정보 확인 : 병사들의 정보를 확인합니다. 
병사 정보 확인으로는 해당 병사의 턴이 종료되지 않습니다.
-재장전 : 병사의 장탄수를 모두 채웁니다.
-게임 체인저 : 강력하고 변수가 많은 랜덤 스킬 하나를 사용하게 됩니다.
 -로켓 런처 : 특정 사분면의 모든 엄폐물들을 파괴합니다.
 -광역 과부하 : 특정 주파수보다 큰/작은 주파수들 모두에 
  특수공격을 할 수 있습니다.
 -시간 왜곡 : 모든 병사들이 턴을 게임 시작시의 순서대로 
  처음부터 진행하게 됩니다.
